# Retro Quest Vault

Una boveda de Obsidian para organizar tareas, proyectos e ideas con estetica de videojuego retro 16 bits.

## Estructura

- `00_HUD/`: panel principal y punto de entrada
- `01_Quests/`: tareas, prioridades y proyectos activos
- `02_Worldbuilding/`: ideas, notas de pensamiento y exploracion
- `03_Systems/`: revisiones semanales y reglas del sistema
- `04_Archives/`: material cerrado o archivado
- `99_Assets/Templates/`: plantillas de uso rapido

## Flujo diario

1. Captura todo en `01_Quests/Inbox.md`
2. Pasa lo importante a `01_Quests/Today.md`
3. Convierte metas grandes en proyectos dentro de `01_Quests/Quest Log.md`
4. Guarda ideas en `02_Worldbuilding/Idea Forge.md`
5. Haz una revision semanal desde `03_Systems/Weekly Reset.md`

## Tema retro

Este vault incluye un snippet CSS en `.obsidian/snippets/retro-16bit.css`.

Activalo asi:

1. Abre el vault en Obsidian
2. Ve a `Settings -> Appearance`
3. Baja hasta `CSS snippets`
4. Pulsa `Reload snippets`
5. Activa `retro-16bit`

## Sincronizacion recomendada

Para usarlo en iPhone y en varias computadoras, la opcion mas estable es `Obsidian Sync`.

Si usas solo ecosistema Apple, tambien puedes probar iCloud Drive. Si mezclas Windows con iPhone, es mejor evitar iCloud y usar Obsidian Sync.
